
# Cheers Dashboard

## libraries used

### Theme - mui-metireal
```
npm install @mui/material @emotion/react @emotion/styled
```

## Tables - x-data-grid
```
npm install @mui/x-data-grid
```

## Icons - mui/icons
```
npm install @mui/icons-matireal
```

## Route - react-route dom
```
npm install react-route-dom localforage match-sorter sort-by
```

## Forms - formik
```
npm install formik
```

## Validation - yup
```
npm install yup
```

## Calender - fullcalender
```
npm install @fullcalendar/core @fullcalendar/react @fullcalendar/daygrid @fullcalendar/timegrid @fullcalendar/list
```

## Charts - nivo
```
npm install @nivo/core @nivo/pie @nivo/line @nivo/geo
